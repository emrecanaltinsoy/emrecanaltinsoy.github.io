# emrecanaltinsoy.github.io

My personal portfolio website built with Jekyll.

## About

This repository contains the source code for my personal portfolio and blog. It showcases my work, resume, and blog posts.

## Acknowledgments

This site uses [snippet-book](https://github.com/jitinnair1/snippet-book) as a reference for theming and structure, which provides Catppuccin color themes with light/dark mode support.

## Development

To run the site locally:

```bash
bundle install
bundle exec jekyll serve
```

Then open your browser at `http://localhost:4000`.
